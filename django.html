Recetas De Cocina

Para Programadores

Framework Django en Ubuntu desde Windows con máquinas virtuales.

Esta receta cubre todo el proceso necesario para poder establecer todo lo necesario para tener un ambiente de desarrollo que permita codificar en Django.

Se encuentra dividido en cuatro secciones. Al completar todas las secciones se podrá contar con lo necesario para iniciar el desarrollo.

Receta para descargar, instalar VMware Workstation Player en Windows:



Ingredientes:

- Navegador Web.
- VMware Workstation Player (Esta receta está hecha sobre la versión 17).

- Tamaño del archivo de instalación: 577 Mb (apróx.)
- Espacio que ocupa la aplicación instalada: 841 Mb (apróx.)

Tiempo De Cocción:

- 10 a 15 minutos.

Requisitos Previos:

- Validar que el equipo cumple con los requisitos mínimos o los recomendados (preferentemente los recomendados).
- El sistema operativo se encuentra actualizado.
- No hay actualizaciones pendientes de instalarse.
- No hay otras aplicaciones ejecutándose, cerrar todas las ventanas innecesarias.

Descarga:

- Desde el navegador web visitar el sitio oficial de descargas de VMware: https://www.vmware.com/es/products/workstation-pro.html
- Deslizarse hacia abajo en busca "Descargar Versión De Prueba" de la versión vigente de VMware Workstation Player para Windows y descargala.

Instalación:

- Hacer doble clic en el archivo de instalación (*.exe) para iniciar el asistente de instalación (siguiente).
- Aceptar los términos de la licencia (siguiente).
- Opcional: Agregar controladores de teclado.
- Dejar tildado "Agregar las herramientas de consola de VMWare Workstation en el PATH de sistema (siguiente).
- Dejar tildado "Buscar actualizaciones del producto al iniciar".
- Opcional: destildar “Unirse al servicio de experiencia del usuario” (siguiente).
- Accesos directos, se puede dejar la configuración predeterminada (siguiente).
- “Confirmar” para iniciar la instalación.
- Se muestra aviso que finalizó la instalación (finalizar).
- Opcional: Reiniciar el sistema (es recomendable hacerlo).

Configuración Inicial:

- Abrir VMware Workstation Player desde el menú de inicio (o desde el ícono en el escritorio).
- Seleccionar uso libre no comercial (continuar).
- "Finalizar" y se abrirá la ventana principal de VMWare Workstation Player.


Se ha completado la receta para descargar, instalar y configurar VMware Workstation Player en sistema Windows.

Receta para descargar imagen ISO de Ubuntu Linux:



Ingredientes:

- Navegador web
- Tamaño del archivo ISO: 4.69 Gb (apróx.)

Tiempo De Cocción:

- 20 a 30 minutos (depende de la velocidad de descarga de la ISO).

Descarga:

- Desde el navegador web visitar el sitio oficial de descargas de Ubuntu: https://ubuntu.com/download/desktop
- Deslizarse hacia abajo en busca de la versión de Ubuntu vigente que se desea descargar. Se recomienda la versión LTS (soporte a largo plazo).
- Al hacer click en el botón "Descarga" se incia proceso de descarga.
- Opcional : Deslizarse hacia abajo en busca de la sección "verificar su descarga" por medio del hash SHA-256 de la ISO.

Se ha completado la receta para descargar imagen ISO de Ubuntu Linux en VMware Workstation Player en sistema Windows.

Receta para crear máquina virtual de Ubuntu Linux:



Ingredientes:

- VMware Workstation Player.
- Imagen ISO Ubuntu Linux.

- La instalación mínima de Ubuntu ocupa 14.5 Gb (apróx.)

Tiempo De Cocción:

- 15 a 20 minutos.

Instrucciones:

- Abrir VMware Workstation Player.
- Elegir la opción "Crear Una Nueva Máquina Virtual".
- Se abre ventana de "Asistente Nueva máquina virtual".
- "Examinar..." para localizar el archivo ISO de Ubuntu Linux.
- Se abre ventana para navegar entre las carpetas para buscar el archivo .ISO de Linux Ubuntu.
- Seleccionar el archivo .ISO (Abrir).
- Hacer click en el botón "Siguiente".
- Completar los campos con los datos de usuario (Siguiente).
- Colocar nombre de la máquina virtual y seleccionar el destino de la nueva máquina virtual. Se puede dejar como aparece por defecto (Siguiente).
- Definir el tamaño del disco duro. Se recomienda 40Gb.
- Dejar tildado "Dividir entre multiples archivos" (Siguiente).
- Para aumentar el tamaño de la memoria ram seleccionar el botón "Editar Hardware".
- Modificar el valor del tamaño de la memoria RAM por 8192 (Cerrar).
- Dejar tildada la opción de "Encender esta máquina virtual luego de la creación" (Finalizar).
- Se dará inicio al proceso de instalación de Ubuntu Linux.
- Distribución Del Teclado. Presionar en el botón "Detectar Configuración Del Teclado". Pedirá presionar algunas teclas para su correcta detección (Continue).
- Elegir la opción de "Instalación Mínima"
- Tildar ambas opciones en "Otras Opciones" (Continue).
- Tipo De Instalación. Dejar seleccionado como se muestra por defecto. Borrar Disco E Instalar Ubuntu. Dejar todo como está (Instalar Ahora).
- Se abre una nueva ventana emergente mostrando "Cambios en el disco" (Continue).
- Elegir la zona horaria (Continue).
- Ingresar Nombre Completo, Nombre De Usuario, Contraseña, Confirmar Contraseña.
- Dejar tildado la opción "Se requiere password para ingresar" (Continue).
- Se mostrará el proceso de instalación del sistema operativo.
- Al finalizar la instalación se abre una nueva ventana emergente (Reinciar Ahora).
- Iniciar sesión.
- Aparece ventana emergente con detalles finales de la instalación.
- Conectar con cuentas en línea (Saltar).
- Habilitar Ubuntu Pro. Tildar la opción "Saltar por ahora" (Siguiente).
- Ayudar a mejorar Ubuntu. Tildar la opción "No, no enviar información del sistema" (Siguiente).
- Privacidad. Dejar desactivada la opción de "Servicios De Localización" (Siguiente).
- Listo para continuar (Listo).
- Puede aparecer ventana emergente solicitando instalar actualizaciones (Instalar Ahora).
- Ingresar contraseña de usuario (Autenticar).
- Al finalizar la actualización solicitará reiniciar el sistema(Reiniciar Ahora).

Instalación de VM-Tools:

- Inciar sesión.
- Abrir "Terminal".
- Ingresar el comando: sudo apt install open-vm-tools (enter).
- Ingresar la contraseña del usuario.

Se ha completado la receta para creación de Máquina Virtual Ubuntu Linux con VMware Workstation Player.

Receta para configurar entorno de desarrollo para django en Ubuntu Linux:



Ingredientes:

- PyCharm
- DBeaver
- Python
- Pip
- Entorno Virtual
- Django

- Espacio que ocupa la aplicación PyCharm instalada: 800 Mb (apróx.)
- Espacio que ocupa la aplicación DBeaver instalada: 259 Mb (apróx.)
- Espacio que ocupa la aplicación Pip instalada: 221 Mb (apróx.)
- Espacio que ocupa la aplicación Entorno Virtual instalada: 4 Mb (apróx.)
- Espacio que ocupa la aplicación Django instalada: 8 Mb (apróx.)

Tiempo De Cocción:

- 10 a 15 minutos.

Requisitos Previos:

- Aplicación Ubuntu Software

Instalar Pycharm:

PyCharm es el IDE (Interface Developer Environment)
- Inciar la aplicación "Ubuntu Software".
- Desde la lupa, realizar la búsqueda de PyCharm.
- Automáticamente se mostrarán resultados de la búsqueda.
- Seleccionar "PyCharm Community Edition".
- Se mostrarán detalles de "Pycharm Community" (Instalar).
- Una vez finalizada la instalación cerrar Ubuntu Software.
- Aparecerá ventana emergente solicitando ingresar contraseña de usuario (Autenticar).
- Iniciar PyCharm para validar que se instaló correctamente.
- Seleccionar "Confirmo que he leído y acepto los términos de este acuerdo de usuario" (Continue).
- Compartir Información (No Enviar).
- Se inciará por primera vez la aplicación, se verá pantalla de Bienvenida.

Instalar DBeaver:

DBeaver es el gestor de base de datos.
- Inciar la aplicación "Ubuntu Software".
- Desde la lupa, realizar la búsqueda de dbeaver.
- Automáticamente se mostrarán resultados de la búsqueda.
- Seleccionar "dbeaver Universal Database Manager and SQL Cliente".
- Se mostrarán detalles de "Pycharm Community" (Instalar).
- Aparecerá ventana emergente solicitando ingresar contraseña de usuario (Autenticar).
- Una vez finalizada la instalación cerrar Ubuntu Software.
- Iniciar DBeaver para validar que se instaló correctamente.
- Aparecerá ventana emergente solicitando crear base de datos de ejemplo (No).
- Se mostrará asistente de base de datos.
- Puede aparecer ventana de "Enviar Información Estadística" (No enviar).

Instalación de Python:

Python viene instalado con Ubuntu.
- Para verificar la versión actual instalada de Python
- Abrir terminal.
- Ingresar el siguiente comando: python3 --version (enter).

Instalación de Pip:

Pip es el manejador de paquetes necesario para instalar Django.
- Abrir terminal.
- Ingresar el siguiente comando: sudo apt install python3-pip (enter).
- Ingresar la contraseña del usuario.
- Solicitará confirmar la instalación (Y).

Instalación del Entorno Virtual:

El entorno virtual es necesario para poder ejecutar un proyecto Django.
- Abrir terminal.
- Ingresar el siguiente comando: sudo apt install python3 virtualenv (enter).
- Ingresar la contraseña del usuario.
- Solicitará confirmar la instalación (Y).

Instalación Django:

- Abrir terminal.
- Ingresar el siguiente comando: pip install Django (enter).

Se ha completado la receta para configurar entorno de desarrollo para django en Ubuntu Linux.
